## What is this?

GrowthLab is a performance-marketing agency based in Monterrey, México. We run paid media (Google Ads, Meta) for B2C and B2B clients across Mexico.

This repository is our **public skill library** — a curated set of Claude Code skills we've battle-tested in real client work. We evaluated 180+ community skills, kept the ones that actually move the needle for marketing and paid media, and ship a few GrowthLab-original skills on top.

We curate and attribute — we do not re-host third-party skills verbatim. Links and attribution for every skill in [`skills/README.md`](skills/README.md).

---

## How to install skills in Claude Code

1. Find a skill you want in [`skills/`](skills/).
2. For **GrowthLab-authored skills** (files in this repo): copy the `SKILL.md` into your project's `.claude/skills/<skill-name>/` folder.
3. For **curated third-party skills**: follow the installation link in [`skills/README.md`](skills/README.md) — install from the original source to respect the author's license.
4. In Claude Code, type `/<skill-name>` to invoke.

> **Tip:** Point Claude Code at your project root and add a `CLAUDE.md` that describes your client/product context. The `gl-*` skills read that file automatically.
