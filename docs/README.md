# Cómo instalar / How to install

## Opción 1: Copiar skills a tu proyecto

1. Clona este repo:
   ```bash
   git clone https://github.com/growthlab-mx/growthlab-skills.git
   ```

2. Copia la skill que quieras al directorio de skills de tu proyecto:
   ```bash
   cp -r growthlab-skills/skills/gl-* ~/.claude/skills/
   ```

3. En Claude Code, invoca la skill:
   ```
   /gl-example-competitive-snapshot
   ```

## Option 2: Add as a Claude Code plugin

1. In Claude Code, run:
   ```
   /plugins add growthlab-skills
   ```

2. Select the skills you want from the GrowthLab library.

# El método GrowthLab / The GrowthLab Method

## Curamos, no reconstruimos

We evaluate 180+ existing skills, keep the ones that work in real paid-media workflows, and share them openly.

**Our approach:**
- Battle-tested in client work (Google Ads, Meta Ads, VBOUT, Oviond).
- Specific deliverables (decks, reports, briefs) — not generic commentary.
- Shareable and on-brand (Spanish language, Tech Blue #1161EE).

## Entrégables impulsados por marca

Every deliverable reflects GrowthLab's identity:
- **Language:** Spanish (Mexico) for client-facing work.
- **Brand:** Tech Blue (#1161EE) primary color; humanist sans font.
- **Format:** Markdown, PPTX, DOCX, or HTML — production-ready.

## Fork, merge, ship

GrowthLab skills build on community foundations (`competitor-profiling`, `ads`, etc.) and add our proprietary process layer.

Use these skills as starting points for your own agency:
1. Fork the repo.
2. Add your brand tokens and deliverable templates.
3. Customize the skill logic to match your process.
4. Deploy to your team.
